# PhotoGallery
Show case of photos demonstrating the the html/css skills

Built on: Google Chrome Version 34.0.1847.116 m
Page title: Code Challenge
Header: ART GALLERY
Scripting Language: HTML5/CSS, Javascript, (CSS3 transition incorporated)
Images resource:lorempixel.com and placekitten.com

Functionality: If clicked on one of the four images, the image enlarges. On clicking on the same image, it returns to it's original position.
Back button: It doesn't have any event on clicking, but it changes color if one hovers over it.
